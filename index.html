paper.css

A minimal, monochrome, classless stylesheet.

'cause sometimes, you just don't need much.

A classless CSS stylesheet. [Article on Medium]
Inspired by omg.lol's Minimal theme by Jane Manchun Wong

My goals were simple, to create something:

  • Simple, no more convoluted frameworks.
  • Lightweight, both on bandwidth and processing power.
  • Pleasant, even if I love some fun sometimes.
Eventually settling on this paper-like design.

"awawawawawa :neofox_floof:"
- Someone.

Want to use it? Feel free to download the stylesheet and link it to your website, like this: 

<link rel="stylesheet" href="/paper.css">
